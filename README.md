## 功能

- 新增任務(透過 plus 圖示或者 Enter 鍵)
- 刪除任務
- 修改任務
- 切換任務狀態(全部、進行中、已完成)
- 一鍵刪除所有任務
- 儲存前一次任務狀態

## 使用的Skills

- 版面與樣式: HTML / SCSS
- 功能: jQuery
